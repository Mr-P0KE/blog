数据结构：线性结构与非线性结构

线性结构

最常用的数据结构，特点：数据元素之间存在一对一的线性关系

存储结构

  1. 顺序存储结构 （顺序表），存储地址连续
  2. 链式存储结构 （链表），存储元素地址不一定联系。

常见的线性结构：数组、队列、链表、栈。

非线性结构

常见的非线性结构：二维数组、多维数组、广义表、树结构、图结构

一、数组

内存结构：

一维数组：

JAVA中数组结构：

  • 8字节的markword
  • 4字节的class指针 （压缩class指针的情况）
  • 4字节的数组大小 （决定了最大容量为 232）
  • 数组元素 + 对齐字节 （java中所有对象大小都是 8字节的整数倍，不足要补齐）

eg：

int[] array = {1,2,3,4,5}
//大小： 8     +     4  +   4       +4*5 +     4 = 40 字节
//    makeword   class     数组大小    数组元素  对齐字节（4 = 24-20）

二维数组：

二维数组： makeword （8） +class （4） +4 （数组大小）+ 4*3 + 4（补齐） = 8+4+4+12+4 = 32

三个一维数组： 8 + 4+ 4 + 4*5 + 4 = 8+4+4+20+4 = 40 *3 = 120

所以一共占用 ： 120+32 =152字节

动态数组：

性能：

  • 随机访问： O（1） （num[x] 会计算内存位置）

  • 修改：O（1）

  • 插入：

    • 在末尾添加元素：平均时间复杂度为 O(1)。当数组已满时，需要进行扩容操作，此时时间复杂度为 O(n)。
    • 在指定位置插入元素：平均时间复杂度为 O(n)。因为需要移动插入位置后面的所有元素。

  • 删除：

    • 删除指定位置的元素：平均时间复杂度为 O(n)。因为需要移动删除位置后面的所有元素。

稀疏数组：

介绍

当一个数组大部分元素为0，或者为同一个值的数组时，可以使用稀疏数组来保存该数组。

eg:

int [][] nums = {{0,0},{1,0},{0,0},{2,0},{3,0}}  //里面有多个重复值0，浪费了内存空间

稀疏数组处理方法

  1. 记录数组有几行几列，有多少个不同的值。
  2. 把具有不同值的元素的行列及值记录在一个小规模的数组中，从而缩小程序的规模。

案例分析：

原数组：6行7列，不同数值有8个

稀疏数组：第零行：保存 共有 6行，共有7列，共有不同数值 8 个

二维数组转稀疏数组：

  1. 遍历原始二维数组，得到有效数据个数sum。
  2. 根据sum创建稀疏数组 spaceArr int[sum+1][y] (sum+1 表示有多少列，第一列为总和，所以sum+1)。
  3. 将二维数组的有效数据存入到 稀疏数组中。

稀疏数组转二位数组：

  1. 读取稀疏数组第一行，根据第一行数据，创建二维数组
  2. 读取稀疏数组后几行，并赋值原始数组。

二、链表

定义：数据的线性存储，空间不连续。

分类：

  • 单向链表

    12 -> 21 -> 25

  • 双向链表

    null <- 12 <-> 21 <-> 25 -> null

  • 循环链表

    12 -> 21 ->25

    |————|

哨兵节点（哑元节点）： 不存储数据，用作 头尾，简化判断

性能：

  • 访问指定位置的元素：平均时间复杂度为 **O(n)**。因为需要遍历链表找到指定位置。

  • 在链表头部添加元素：时间复杂度为 **O(1)**。

  • 在链表尾部添加元素：对于单向链表，**时间复杂度为 O(n)；对于双向链表，时间复杂度为 O(1)**。

  • 在指定位置插入元素：平均时间复杂度为 **O(n)**。因为需要遍历链表找到插入位置。

  • 删除指定位置的元素：平均时间复杂度为 **O(n)**。因为需要遍历链表找到删除位置。

        }
        return  null;
    }

        Node temp = new Node(val,head,pre);
        head.next = temp;
        pre.prev = temp;

    }

        remove(0);
    }
    public void addLast(int val){
        Node pre = tail.prev;
        Node temp = new Node(val,pre,tail);
        pre.next = temp;
        tail.prev =  temp;
    }
    public void removeLast(){
        Node pre = tail.prev;
        if(pre == head)
            throw illegalIndex(0);
        Node temp = pre.prev;
        temp.next = tail;
        tail.prev = temp;
    }
    public void insert(int index,int val){
        Node temp = findNode(index-1);
        if(temp == null )
            throw illegalIndex(index);

        Node next = temp.next;
        Node n = new Node(val,temp,next);
        temp.next = n;
        next.prev = n;
    }

        Node pre = findNode(index-1);
        if(pre == null || pre == tail || pre.next == tail)
            throw illegalIndex(index);

        Node next = temp.next;
        pre.next = next;
        next.prev = pre;
    }

数组模拟环形队列（取模运算判断元素个数）

在环形队列中，我们要判断是否为空或者为满，需要提前预留一个空位。

所以容量为n的环形队列，只能存储 n - 1个 数据。

思路（有多种）：

取模：

front 变量 ：指向队列的第一个元素。 初始值 0

rear 变量： 指向队列的最后一个元素的后一个位置。（空出一个空间做约定（判断为空，或者为满）），初始值 0

当队列满时，条件为 ： （rear + 1） % maxSize = front （eg : maxSize 为10，那么rear最大为9，所以要+1取模）

当队列为空，条件为： rear = front

队列中有效的元素个数： (rear + maxSize - front) % maxSize (rear + maxSize是为了让rear始终大于front，eg: rear已经是第二圈了，但是rear = 2 ，而front = 3，所以需要 +maxSize )

        this.capacity = capacity+1;

        if(i+1 >= length)
            return i = 0;
        return ++i;

        if(i-1 < 0)
            return i = length-1;
        return --i;
    }
    @Override
    public boolean offerFirst(E e) {
        if(isFull())
            return false;
        l = dec(l,capacity);
        array[l] = e;
        return true;

    private int size;

    public PriorityQueue(int capacity) {
        priorities = new Priority[capacity];
    }

    @Override
    public boolean offer(E value) {
        if(isFull())
            return false;
        priorities[size] = value;
        size++;
        return true;
    }

    /**
    * @Author Mr_Poke
    * @Date 12:03 2023/5/7
    * @Description 返回优先级最高的索引值
    * @Param  * @param
    * @Return int
    */
    private int findMax(){
        int max = 0;
        for(int i = 1;i<size;i++){
            max = priorities[max].priority() > priorities[i].priority() ? max : i;
        }
        return max;
    }
    @Override
    public E poll() {
        if(isEmpty())
            return null;
        int max = findMax();
        E e = (E) priorities[max];
        remove(max);
        return e;
    }
    private void remove(int index){
        if (index < size - 1)
            System.arraycopy(priorities,index+1,priorities,index,size-index+1);
        priorities[--size] = null;
    }
    @Override
    public E peek() {
        if(isEmpty())
            return null;
        return (E) priorities[findMax()];
    }

    @Override
    public boolean isEmpty() {
        return size == 0;
    }

    @Override
    public boolean isFull() {
        return size == priorities.length;
    }

               //复制右子树
               int[] inright = Arrays.copyOfRange(in, i+1, in.length);
               
               //复制下一个节点左子树
               int[] preleft = Arrays.copyOfRange(pre, 1, i+1);
               //复制下一个节点右子树
               int[] preright = Arrays.copyOfRange(pre, i+1, pre.length);
               
               root.left = buildTree(preleft, inleft);
               root.right = buildTree(preright, inright);  
               break; 
           }
       }
       return root;
   }

104：二叉树深度

深度问题是典型的考察二叉树的后序遍历。

方法：深度 = Max （左子树高度 ， 右子树高度） + 1

递归思路：
  • 递 到第一个叶子节点， return 0;
  • 归 到第一个根节点（第一个叶子节点的根节点），l = 0,r =0, return 0+1=1;
public int maxDepth(TreeNode root) {
       if(root == null)
       return 0;
       int l = maxDepth(root.left);
       int r = maxDepth(root.right);
       return Math.max(l,r) + 1;
   }
