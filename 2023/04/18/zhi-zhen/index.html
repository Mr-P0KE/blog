指针

一、内存

物理内存和虚拟内存

虚拟内存：操作系统虚拟出来的内存

32位电脑 32根寻址总线

0x00 00 00 00

0xff ff ff ff

操作系统会在物理内存和虚拟内存之间做映射

在32位系统下，每个进程的寻址范围是4G，0x00 00 00 00 ~0xff ff ff ff

应用程序中，我们看到的是虚拟地址

运行程序时，操作系统将虚拟内存进行分区：

虚拟内存分区

  1. 堆

    在动态申请内存时，在堆中开辟内存

  2. 栈

    主要存放局部变量

  3. 静态全局区

    1.未初始化的静态全局区

    静态变量（定义变量，static修饰），全局变量，没有初始化。

    2.初始化的静态全局区

    全局变量、静态变量、赋过初值、

  4. 代码区

    存放程序代码

  5. 文字常量区

    存放常量

内存以字节为单位存放数据

二、指针的概念

系统给虚拟内存的每个存储单元分配编号：0x00 00 00 00 ~0xff ff ff ff

编号称为地址

指针就是地址

指针（本身占4个字节，存储值的内存地址）–根据指针存储的内存地址（值的内存地址）–找到值

int *p;
//p为指针 *p为指针存储的值

指针变量：一个变量，存放地址编号

注意：

  1. 任何类型的指针变量在32位系统下都是4个字节，在64位占据8个字节,

  2. 对应类型的指针变量，只能存放对应类型的指针地址

    eg：整型变量的指针，只能存放指针变量的地址

  3. 多字节变量的地址值有多个，系统默认最小的编号为指针。

三、指针的定义方法

1.简单的指针变量

数据类型 * 指针变量名；
int *p; //定义了一个指针变量

*用来修饰变量，说明变量p是一个指针变量，变量名为p

注意：

int *p,*x;

必须都有*，用来表示指针

2.指针的运算符

& 取地址

* 取值 : temp = a

int main() {
    int *p;
    int a = 10;
    p = &a;

    printf("%p\n",&a); //获取a的地址
    printf("%p\n",p); //获取指针指向的a的地址  p = &a,所以地址一致
    printf("%p\n",&p);//获取指针的地址
    printf("%d\n",*p);//获取指针指向的a值

}

p与a的关系分析：

a为0x1234abcd，假设a的地址为0xbf e8 98 68

五、指针和变量的关系

指针的地址不会改变，指针地址中存储的值（指针指向的值的地址，也就是 *p）可以改变

#include <stdio.h>

int main() {
    int a,b;
    int* p1=&a,*p2= &b;
    printf("请输入a和b的值\n");
    scanf_s("%d %d",p1,p2); // 1 2
    printf("%p %p\n",p1,p2); //0000005b6d7ffac8 0000005b6d7ffac4

    int temp = *p1;
    *p1 = *p2;
    *p2 = temp;
    
    printf("%p %p\n",p1,p2); //0000005b6d7ffac8 0000005b6d7ffac4

    printf("%d  ",a); // 2 1
    printf("%d\n",b);
    printf("%d  ",*p1);// 2 1
    printf("%d\n",*p2);

}

只要字符型变量值的二进制第一位是1，就会如buf[0]所显示的那样，出现6个f。

int main() {
   int a =  0x1234678,b = 0xabcdef66;
   char* p1,*p2;
   int*p;
   p1 = (char *) &a;
   p2 = (char *) &b;
   printf("%0x %0x\n",*p1,*p2); //78,66
   p = &a;
   printf("%0x\n",*p);//12345678，因为p是int类型的可以存储4个字节
   p1++; //指针++
   p2++;
   printf("%0x %0x\n",*p1,*p2); //46（因为是有符号数自动补全，但是舍去0） ffffffef(因为是有符号数，自动补全)
    printf("%0x %0x\n",*p1,0xff & *p2);// 与运算（高位变0，舍去）
}

格式输出函数printf中会对所输出的变量做有符号/无符号型的判断。如果是有符号型变量，且该值二进制首位为1（如我们定义的是char buf[100]，且buf[0]中为0xbf），则会按照补码形式前面全置为1，也就是全为f。

六、指针与数组的关系

1.数组引用方法：

  1. 数组名【下标】：

    int a[5] ; 
a[2] = 100

  2. 指针名【下标】：

    int a[5];   
int *p;   
p = a  ;   
p[2] = 100

C语言规定，数组名字为数组的首地址，即第0个元素的地址，就是&a[0],是常量

注意！：p与a不同，p是指针变量，而a是常量，所有可以用 = p赋值，而不能给a赋值。


p = &a[3]; //正确，因为p是变量
a = &a[3];  //错误，因为a是常量

  1. 指针变量运算来引用数组元素

    int a[5];
int *p;
p = a;
*(p+2) = 100; //等价于 a[2] = 100

  2. 通过数组名 + 取值引用数组元素

    int a[5];
*(a+2) = 100 ////等价于 a[2] = 100

2.指针的运算

1.指针可以加一个整数，往下指几个它指向的变量，结果还是地址

前提：指针指向数组

int a[5] = {0,1,2,3,4,};
   int *p;
   p = a;
   p+2; // 等价于 &a[2]  加了8个字节
char c[5] = {'a','b','c','d','e'};
   char *p;
   p = c;
   p =p+2; // 等价于 &a[2]  加了2个字节

2.两个相同类型指针比较大小

前提：两个指针类型相同、指向同一个数组的元素，比较才有意义

int main() {
   int nums[5];
   int *p1,*p2;
   p1 = &nums[1];
   p2 = &nums[2];
   if(p1 < p2)
       printf("jajaj"); //输出
   else
       printf("hahah");
}

3.两个相同类型的指针做减法

前提：两个指针类型相同、指向同一个数组的元素，比较才有意义

目的： 计算两个指针元素中间包含几个元素

int nums[5];
   int *p1,*p2;
   p1 = &nums[1];
   p2 = &nums[4];
    printf("%d",p2-p1); // 4-1 = 3
    printf("%x\n",p1);  //f3bff744
    printf("%x\n",p2); //f3bff750 相差12个字节（3位* int的4个字节）

4.相同类型指针相互赋值

只有相同类型指针才可以相互赋值（void * 除外）

int *p1,*p2;
int a;
p1 = &a;
p2 = p1;

注意：如果类型不相同的指针要相互赋值，必须强制类型转换。

八、指针的指针

指针的指针，即指针的地址

32位系统中，指针占4个字节。64位系统中。指针占8个字节。

//eg：
int a = 0x123445678;
//假设i的地址为 0x00002000
int *p;
p = &a;      //存放a的地址
int **q;
q = &p;     //存放指针p的地址
int ***m;
m = &q;      //存放指针q的地址
printf("%d\n",*p);
printf("%d\n",**q);
printf("%d\n",***m);

以32系统为例

a值为0x12345678，占据4个字节（32位系统），地址的最低位为0x00002000，

指针p存储a的最低位地址0x00002000，p的内存地址最低位为0x00003000,

指针q存储指针p的内存最低位0x00003000，q指针内存地址的最低位为0x00004000，

指针m存储指针q内存地址的最低位0x00004000，m指针的内存最低位为0x00005000.

九、字符串和指针

1.什么是字符串？

以\0结尾的若干个字符的集合。

字符串的地址为第一个字符的地址。

eg： “Hello World!” 的地址，实际上是’H’的地址。所以定义指针： char *c = “hello world”;

2.字符串的存储形式

数组、文字常量区、堆

1.存储在数组中

存储在内存中（栈、静态全局区）

char string[100] = " I love C!";

注意：

普通全局数组 ： 存放静态全局区

普通局部数组 ： 存放栈区

静态数组（静态全局数组、静态局部数组）：存放在静态全局区

2.存放在文字常量区

char* str = "I love C!";

定义了一个指针变量str，用来存放字符地址编号，

"I love C!"存放在文字常量区

3.存放在堆区

使用malloc等函数在堆中申请空间，将字符串拷贝在堆区

char *str = (char*)malloc(10); //动态申请10个字节的存储空间，首地址给str赋值
strcpy(str, "I love C!"); //将字符串 "I love C!"拷贝到str指向的内存中

3.字符串的可修改性

1.存放在数组中可修改

int main() {
   char string[100] = "Hello World!";
    printf("%s\n",string);  //Hello World!

    string[0] = 'W';
    printf("%s\n",string);  //Wello World!
    return 0;
}

注意：无 const （只读）修饰的数组才可以修改

2.存放在文字常量区不可修改

char *str = "I love C!";
printf("%s\n",str);
//    *str = 'w'; //错误
str = "wo men shi yi jia ren"; //正确
printf("%s\n",str);

3.存放在堆区可以修改

 

int main() {
    char *str = (char*)malloc(10); //动态申请10个字节的存储空间，首地址给str赋值
    strcpy(str,"I love C!"); //将字符串 "I love C!"拷贝到str指向的内存中
    printf("%s\n",str);
    *str = 'Y';
    *(str+1) = 'Y';
    printf("%s\n",str); //YYlove C!
}

4.赋值

1.在数组中赋值

int main() {
   char str[100] = "Hello World!";
    str = "123333";  //错误，因为字符数组的名字是常量，无法使用等号赋值
    strcpy(str,"Hello 123");
    printf("%s\n",str);
    scanf("%s",str);   //输入值，并修改保存到str中
    printf("%s\n",str);
}

2.在文字常量区

char * str = "Hello World!";
str = "123456";         //正确，指向了另一个字符串
strcpy(str,"1233333");// 错误，因为文字常量区只读，不能通过指针修改文字常量区内容

3.指向堆区

char* str;
str = (char *)malloc(15);
strcpy(str,"123333");
scanf("%s",str);

总结：

  • 指针可指向文字常量区

    • 指针指向的文字常量区内容不可修改

    • 指针的指向可改变，即可以给指针重新赋值。

  • 指针可指向堆区

    • 指针指向的堆区内容可以修改。
    • 指针的指向可以改变，可以给指针重新赋值

  • 指针可以指向数组（非const修饰）

    char str[15] =  "Hello World!";
char *s = str;
*(s+1) = 's'; // e修改为s
printf("%s\n",str);
//    str = "123"; //错误
s = "123456";
printf("%s\n",s); //指针s重新指向“123456”

十、数组指针

1.二维数组

int a[3][5];
   printf("%p\n",a); //0000005116dff8e0
   printf("%p\n",a+1); //0000005116dff8f4
                        //相差20个字节，（int占4个字节，4+5 = 20）

2.数组指针的概念

本身是一个指针，指向一个数组，+1指向下一个数组。

3.定义数组指针

指向的数组类型 （* 指针变量名） 【指向的数组元素个数】

int (*a) [5];
int main() {
    int  a[3][5];
    a[0][0] = 20;
    int (*p) [5];
    p = a;
    printf("%p\n",p);
    printf("%p\n",a);  //相等

    printf("%p\n",a+1);
    printf("%p\n",p+1);   //相等

    printf("%d\n",*p[0]); //20

}

常用于数组传参

int function(int (* p)[5],int x,int y);
int main() {
    int  a[3][5] = {{1,2,3,4,5},
                    {6,7,8,9,0,},
                    {11,12,13,14,15}};

    function(a,3,5);
}
int function(int (* p)[5],int x,int y){
    for (int i = 0; i < x; ++i) {
        for (int j = 0; j < y; ++j) {
            printf("%d ",p[i][j]);
        }
        printf("\n");
    }
    return 0;
}

  1. 一维数组指针

    配合二维数组使用

    int (*p) [5]; // 与数组长度一致
int a[3][5];
p = a;

  2. 二维数组指针

    int (*p) [4][5];
int a[3][4][5];
p = a;

  3. 多维数组指针

5.指针数组与数组指针的区别

  • 指针数组：一个数组，若干个相同的指针构成的集合

    int *p[10];
//数组p有10个int*类型的指针变量，p[0]~p[9]

  • 数组指针：本身是一个指针，指向一个数组，+1 跳一个数组（多维数组）

    int (*p) [10];
//p是一个数组指针，p+1跳下一个数组

  • 指针的指针

    int **p;
int *q;
q = &p;

6.数组名字取地址：变成 数组指针

一维数组名字取地址，变成一维数组指针，即 +1 跳一个一维数组

int a[10];
a +1; //a[1]的地址
&a  // int (*p) [10] ,变成了一维数组指针
int a[10];

printf("%p\n",a);   //int * 类型的指针，指向a[0]地址
printf("%p\n",&a); //int (*p) [10]数组指针  两者相等，但是类型不一致 

printf("%p\n",a+1);  // a与 a+1相差4个字节
printf("%p\n",&a+1);  // &a 与 &a+1 相差  40个字节
return 0;

二维数组名字取地址，变为二维指针，即 +1 跳一个二维数组

int a[4][5];
printf("%p\n",a);     //int (*p) [5]
printf("%p\n",a+1);  //相差20个字节，即一个一维数组

printf("%p\n",&a);   //int (*p) [4][5]
printf("%p\n",&a + 1); //相差80个字节，即一个二维数组
return 0;

一维数组 ：a+1 指向下一个元素。 &a+1 指向下一个一维数组

二维数组： a+1 指向下一个元素（数组），&a+1 指向下一个二维数组

7.数组名字与指针变量的区别

int a[10];
int *p;
p = a;

int (*pp) [5];
pp = &a;

相同点：

  1. 在引用时等价。

    a 是数组名字，保存 a[0] 地址, p = a

    p 保存了a[0] 地址

      a[2] = *(a+2) 
= p[2] = *(p+2)

    举例：

    int main(){
    int a[5] = {0,1,2,3,4};
    int *p;
    p = a;
    
    printf("%p\n",a);
    printf("%p\n",p);  // a = p
    
    printf("%d\n",a[2]); //2
    printf("%d\n",p[2]); //2
    printf("%d\n",*(a+2));//2
    printf("%d\n",*(p+2));//2
}

不同点：

  1. a是常量 ， p是变量

    无法使用 = 给a赋值。但是p可以。

    int a[5] = {0,1,2,3,4};
   int *p;
   p = a;
   
   p = &a[3];
   printf("%d\n",*p);
   
   //a = &a[3];  //错误

  2. &a 与 &p 不同

&a &p
类型 数组指针 ( int (*p) [5] ) 指针的指针 ( int **p 类型)
+1 &a+1 ：往后指一个数组 (相差20个字节) &p+1 ：往后指一个int* 类型（相差4个字节） 
int a[5] = {0,1,2,3,4};
int *p;
p = a;
   
//p = &a[3];        //正确
printf("%p\n",&p); // 指针 p 的地址，
   
printf("%p\n",&a); // 常量 a 的地址。 p = &a
int main(){
    int a[5] = {0,1,2,3,4};
    int *p;
    p = a;

    int (*pp) [5];
    pp = &a;

    printf("%p\n",a);    // p = a
    printf("%p\n",a+1); //相差4个字节
    printf("%p\n",&a);  // pp = &a
    printf("%p\n",&a+1); //数组指针，相差 20 字节

    printf("%p\n",&p);    
    printf("%p\n",&p+1); //指针的指针，相差 4 个字节（win64位，相差8）

}
